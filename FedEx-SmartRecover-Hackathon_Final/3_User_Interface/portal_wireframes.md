# DCA Portal Wireframes

## 1. Login Screen
- Role-based authentication
- MFA required for all external agencies

## 2. Dashboard View
- Cases assigned to the agency
- Urgent cases highlighted (24h SLA)
- Performance metrics visible

## 3. Case Detail View
- Tokenized customer ID (FED-XXXXX)
- P2P score and confidence indicator
- Communication log (calls, emails, promises)
- Dispute escalation button

## 4. Reporting View
- Weekly recovery reports
- Commission calculations
- Performance analytics
